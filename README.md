1. Install Required Software
    The game DUST.
    MelonLoader from GitHub (version 0.7.2)
   
2. Prepare the Game
    Open the game at least once, then close it.
    In MelonLoader, select Dust.exe
   
3. Install MelonLoader
    Click Install and wait for it to finish.
    Launch the game: a console will appear — this is normal.
    Watch the console: the installation of .NET 6 should begin. If there’s an error, download and install .NET 6 from the official website.

4. Install the Mod
    Download the mod .dll file and place it in the Mods folder.
    Launch the game and open the console. If the mod is working, you will see information about it. Errors like OpenXR can be ignored — they     do not affect mods.  
    
5. Verify Mod Functionality
    The console should display information about the mod.
    If no information appears, go back and check the previous steps to ensure everything was done correctly.
